# HTTP Monitor

Full-stack application that pings `httpbin.org/anything` every 5 minutes with a random JSON payload, stores the responses in MongoDB, and displays them on a real-time dashboard with LLM-powered insights.

Built for the BizScout Full Stack Engineer take-home test.

> Documentation (setup, architecture, decisions, testing strategy) is filled in as the project is built — see commit history for the development journey.
